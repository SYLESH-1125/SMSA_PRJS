Demo Free Fire (Next.js + Supabase)

What this is:
- A small Next.js demo showing a Free Fire-like minimal canvas game (move + shoot) with score saving to Supabase and a leaderboard.

Run locally:
1. npm install
2. Create .env.local with NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY
3. npm run dev

Notes:
- Supabase calls use placeholders; do not commit secrets. Test locally with your Supabase project.
- Use demo-script.json to record the demo for YouTube.
